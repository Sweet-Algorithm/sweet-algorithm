# sweet-algorithm 🎁💝

## 🗒️ branch 사용방법
1. 해당 repo를 fork 뜬다.
2. fork 떠온 개인 repo에서 하루마다 문제를 푼다.
3. 최종 다 푼 문제는 매일 sweet-algorithm branch에 pr를 날린다.
   (다른 사람도 함께 볼 수 있도록!!)
4. pr의 merge는 매일 자정이 넘어가기 10분전쯤 진행한다. => 해당 pr에 리뷰를 달수 있음 !
